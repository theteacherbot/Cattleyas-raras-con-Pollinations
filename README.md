# 🌺 Cattleya · Especies Raras

Galería fotográfica interactiva dedicada a las orquídeas del género **Cattleya** de distribución restringida y rareza. Diseño editorial elegante con experiencia de usuario fluida.

![Vista previa](preview.png)

## ✨ Características

- **5 especies documentadas**: C. granulosa, C. schilleriana, C. tenebrosa, C. praestans, C. warneri
- **Filtros dinámicos** por categoría taxonómica
- **Lightbox integrado** con navegación prev/siguiente
- **Diseño responsive** optimizado para móvil y desktop
- **Paleta cromática sofisticada** inspirada en tonos naturales de orquídeas
- **Tipografía editorial**: Cormorant (serif) + Figtree (sans-serif)

## 🛠️ Tecnologías

| Aspecto | Implementación |
|---------|---------------|
| Estructura | HTML5 semántico |
| Estilos | CSS3 custom properties, Grid, Flexbox |
| Interactividad | Vanilla JavaScript (sin frameworks) |
| Tipografías | Google Fonts (Cormorant, Figtree) |
| Imágenes | Base64 embebido + placeholders |
